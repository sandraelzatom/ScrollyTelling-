# Specification: Life Beyond the Screen
**Author:Sandra Elza Thomas
**Date: 23 April 2026

## 1. Project Goal
To create a high-impact "Scrollytelling" website that introduces the user to my hobbies using a fixed-background scrolling effect.

## 2. Content Structure (The Storyboard)
- **Slide 1: Intro**
  - Text: "Life Beyond the Screen"
  - Purpose: Catch the user's attention.
- **Slide 2: Hobby - Gaming**
  - Text: "Leveling up and strategy."
  - Asset: Background image of a controller or setup.
- **Slide 3: Hobby - Cooking**
  - Text: "The art of flavor."
  - Asset: Background image of a kitchen or meal.
- **Slide 4: Hobby - Basketball**
  - Text: "Staying active on the court."
  - Asset: Background image of a basketball hoop.

## 3. Technical Requirements (Spec for AI)
- **Framework:** Vanilla HTML5 and CSS3.
- **Effect:** Use `background-attachment: fixed` to create the scrollytelling feel.
- **Theme:** Dark mode (Background: #111, Text: white).
- **Layout:** Each section must be exactly `100vh` (full screen height).
